E-Voting System

Full-stack E-Voting app with Spring Boot backend, React frontend, and MySQL database.

Features

Add users, elections, and choices.
Cast votes and get election winner

Setup

Backend: Configure MySQL in application.properties → mvn spring-boot:run.
Frontend: cd voting-frontend → npm install → npm start
Runs at http://localhost:3000.
Backend default port: 8082.
